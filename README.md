# Yunlei Shen — Portfolio

Static portfolio site. No build step — plain HTML + JS.

## Structure

```
index.html                        Home (work overview)
about.html                        About
the-end-of-the-paper-trail.html   Case study — Valor claim digitization
a-crm-worth-navigating.html       Case study — Sugar CRM re-architecture
four-roles-one-system.html        Case study — Sugar CRM role workflows
the-two-minute-deal.html          Case study — influencer-business matchmaking
assets/
  images/                         All image assets
  js/                             Page runtime (support.js)
  docs/                           Résumé PDF
```

## Deploy

Works as-is on GitHub Pages: push this folder to a repo, then enable
Pages (Settings → Pages → Deploy from branch → root). `index.html`
is the entry point.
